# Sales Data ETL Pipeline

##  Objective
Build a basic ETL pipeline to process sales data.

## 🔧 Steps Performed

Extract
- Loaded CSV data using pandas

Transform
- Handled missing values
- Converted data types
- Created new column (total_amount)
 Load
- Saved cleaned data into new CSV

 Technologies Used
- Python
- Pandas
- Linux (PuTTY, WinSCP)

 Output
- Clean dataset: sales_cleaned.cs
